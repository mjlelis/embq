All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-19

### Added
- Initial release.
- Core kernel with fp32, int8, and binary quantization.
- PyO3 bindings for Python integration.
- Python CLI for easy profiling.
- GitHub Actions CI for multi-platform wheel builds.
